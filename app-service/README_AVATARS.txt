# PluralMatrix Avatar Recovery Guide

Your system data has been exported from PluralMatrix in a PluralKit-compatible format. 
Because PluralKit requires public HTTPS links for avatars, we have included your avatar images in the 'avatars/' folder and set placeholder URLs in the JSON file using 'myimageserver'.

To restore your avatars in Discord/PluralKit, use one of the following methods:

### Method 1: Manual Update (Easiest for small systems)
1. Import the 'pluralkit_system.json' file into PluralKit as usual.
2. For each member, use the Discord command 'pk;member <name> avatar' and upload the corresponding image from the 'avatars/' folder.

### Method 2: Custom Image Server (Best for large systems)
1. Upload the entire 'avatars/' folder to a public web server or hosting service.
2. Open 'pluralkit_system.json' in a text editor.
3. Use "Find and Replace" to replace 'https://myimageserver/avatars/' with your actual server URL (e.g., 'https://yourdomain.com/my-avatars/').
4. Import the modified JSON file into PluralKit.

### Method 3: Third-Party Image Hosting
1. Upload each image in the 'avatars/' folder to a service like Imgur, Catbox, or Litterbox.
2. Update the 'avatar_url' fields in the JSON file with the new links before importing, or use 'pk;member avatar <link>' after importing.
